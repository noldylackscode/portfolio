🎓 Thesis Project – Student Intervention Monitoring System

🏫 School: Trimex Colleges

👨‍💻 Role: Co-Developer (Backend & Frontend)

🛠️ Technologies: Visual Basic, MySQL, PHPMailer, RFID System

📌 Project Overview

Developed in collaboration with a teammate, this system automates student monitoring using an RFID-based attendance and grading system. Students who are absent for three consecutive days or have an average below 75 in major subjects are flagged as "At Risk," labeled as "Subject for Improvement" in our research documentation.

We built the system entirely using VB.NET for both the frontend and backend, with MySQL as the database. Email alerts are automatically sent via PHPMailer when specific risk conditions are met. The system also tracks student intervention statuses such as "At Risk," "Subject for Improvement," "Failed," or "Intervention Completed."

Key features include teacher-subject management, grade input, student performance tracking, searchable records, and full CRUD functionality.

🧾 Cashier and Inventory System – CRUD & Barcode Project

🏫 School: Trimex Colleges

🗓️ Duration: 2nd Year College (School Presentation)

👨‍💻 Role: Solo Developer

🛠️ Technologies: Visual Basic, MySQL

📌 Project Overview

A simple yet functional cashier and inventory system built as a CRUD-focused presentation project. Designed to replicate real-world sales systems like those in Super8 or mini-marts.

Features include barcode scanning, automatic sales computation, and product deduction from inventory after each successful transaction.

The system allows full CRUD functionality for product inventory and sales records. Items marked as sold are automatically reflected in the sales database, and inventory counts are updated accordingly.

Though built primarily for demonstrating CRUD logic, the system provides a clear simulation of real-world cashier workflows.
